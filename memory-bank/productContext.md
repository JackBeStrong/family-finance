# Product Context - Family Finance Manager

## Project Overview
A Python-based family finance management system designed to help track, analyze, and plan household finances.

## Core Goals
1. **Bank Statement Processing** - Read and parse bank statements/transactions (weekly/monthly)
2. **Financial Reporting** - Generate summary reports of income, expenses, and trends
3. **Home Loan Management** - Track mortgage payments, interest, and payoff projections
4. **Investment Tracking** - Monitor investments and provide suggestions
5. **Long-term Planning** - Financial goal setting and progress tracking

## Target Users
- Family members managing household finances
- Primary focus on Australian banking formats initially

## Key Features (Planned)
### Phase 1 - Bank Statement Reader (Current)
- CSV/PDF bank statement parsing
- Transaction categorization
- Weekly/monthly summary reports
- Expense breakdown by category

### Phase 2 - Home Loan Management
- Mortgage tracking
- Extra payment calculations
- Interest savings projections

### Phase 3 - Investment & Planning
- Investment portfolio tracking
- Goal-based savings plans
- Long-term financial projections

## Technical Stack
- **Language**: Python 3.x
- **Data Storage**: SQLite for local data, CSV for imports/exports
- **Reporting**: Pandas for data analysis, potential for visualization libraries
